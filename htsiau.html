Install and use fss.dll

  1. Download the DLL from this page.
  2. Press Windows + R, type powershell and press Enter.
  3. Navigate to the download folder:
    cd C:\Path\To\DLL
  4. Load the DLL:
    Add-Type -Path .\fss.dll
  5. Use the functions like this:
    [fss.Class]::YourFunction()
