fss.dll

  1. Download the DLL from here.
  2. Press Windows + R, type powershell and press Enter.
  3. Navigate to the folder where the DLL is located using:
    cd C:\path\to\the\dll
  4. Load the DLL using:
    Add-Type -Path .\fss.dll
  5. You can now use the DLL in PowerShell like this:
    [fss.Class]::Func()
