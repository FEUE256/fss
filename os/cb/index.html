About

Cyber:Bott OS Edition is a bold and independent operating system developed from scratch by FÈUE FSS. Designed exclusively for modern UEFI systems, it represents the future of OS innovation (a Hello World BIOS version is also available).

Roadmap

  • Display "Hello World" on screen
  • Build a stable system API
  • Develop a graphical shutdown interface
  • First public release.

Features in CyberBoot UEFI

  • Informational GUI
  • Test Shutdown GUI
  • Custom API for FÈUE FSS Cyber:Bott OS Edition
